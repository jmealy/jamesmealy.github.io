Final Year Project

Sentiment Analysis and Topic Modelling in Python

The aim of this project was create a desktop application (similar to a file manager) capable of labeling files according to their sentiment using machine learning techniques. This involves taking a list of labelled training data to create a classifier, which was then used to assign positive or negative sentiment to unlabeled files.

The training data used for testing this project was a set of movie reviews labeled as positive or negative. Positive or negative sentiment was inferred from the 5 start rating given to the movie by the review.

The project was then extended to be able to classify files by language, topic, or any other criteria, as long as the appropriate training data was provided.

The poject was extended further to be able to classify files according to their topic without needing labeled training data, ie. unsupervised classification. This was done using a topic modelling algorithm called Latent Dirichlet Allocation.

The main Python libraries used in this project were:

  • Scikit-learn
  • NLTK
  • Gensim
  • wxPython
  • NumPy

The source code and formal report for the project can be seen on Github.
